# etch-a-sketch

A basic web application for The Odin Project, that contains a sketching grid with various drawing functionalities. Built with fundamental HTML, Flexbox and JavaScript.

## Live Demo

https://nat0134.github.io/etch-a-sketch/

## About The Project

This project demonstrates basic front-end web developement skiils using fundamental Hypertext Markup Language, Cascading style sheet and Flexbox layout module. The application dynamically generates grid elements through DOM manipulation and features an interactive sketchpad with multiple drawing functionalities.

It presents an understanding and use of:
- Semantic HTML structure
- CSS rule application for styling and layout
- Precision control of HTML elements using cascading properties
- Applying Flexbox layout module to responsive grid structure
- Dynamic element generation through DOM manipulation
- Handling user interaction with event listeners
- Managing application state for drawing modes and canvas resizing
